O que isto faz

Quando habilitado, o cockpit faz dual-write: tudo que é gravado no SQLite local (72h de retenção) é também replicado para um PostgreSQL remoto, onde fica disponível pelo tempo que você definir (default: 30 dias). Cada tabela do PG ganha o prefixo do hostname para permitir que vários hosts apontem para o mesmo banco sem conflito.

  • Tabela local metrics_cpu → tabela remota {{ data.cfg.table_prefix }}_metrics_cpu
  • Falhas no PG não bloqueiam a gravação local.
  • Senha armazenada em SQLite local (file mode 600). Use rede privada ou TLS para o PG.
